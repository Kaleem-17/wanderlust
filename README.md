# Wanderlust

A full-stack travel listing platform where users can discover, share, and review destinations worldwide.

## Tech Stack

- **Backend:** Node.js, Express.js, MongoDB, Mongoose
- **Frontend:** EJS, Bootstrap 5, JavaScript
- **Authentication:** Passport.js (Local Strategy)
- **APIs:** MapTiler (maps), Cloudinary (image upload)
- **Deployment:** Ready for Render, Cyclic, or Railway

## Features

- User registration & authentication
- Create, read, update, delete travel listings
- Upload multiple images per listing
- Add, edit, and delete reviews
- Search listings by location or country
- Interactive maps for each destination
- Responsive design for all devices
